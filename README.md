# Click Links Continued

**Click Links Continued** is a lightweight World of Warcraft addon that makes URLs clickable in chat.

Maintained fork with modern API support, multi-client compatibility, and an optional **Link Journal**.

## Features
- Clickable URLs (websites, emails, IPs)
- Safe copy popup
- Link Journal with cap protection
- Full AceLocale coverage (12 languages)
- Minimap button
- Version checking
- Retail, Classic, TBC, Wrath, Titan Forged support

## Commands
/cl journal — Toggle Journal
/cl minimap — Toggle minimap button
/cl version — Print version
